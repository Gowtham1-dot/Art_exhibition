# ARTT Frontend (Next.js)

This folder contains the Next.js UI for ARTT.

For full project setup (backend + frontend + env vars), see the root README:

- `../README.md`

## Run locally

Create `frontend/.env.local`:

```bash
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

Start dev server:

```bash
npm install
npm run dev
```
